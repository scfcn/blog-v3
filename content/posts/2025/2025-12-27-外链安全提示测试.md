# 外链安全提示测试

本文用于测试外链安全提示组件的功能。

## 外部链接测试

以下是一些外部链接，点击它们应该会触发安全提示弹窗：

- [Google](https://www.google.com) - 搜索引擎
- [GitHub](https://github.com) - 代码托管平台
- [Bilibili](https://www.bilibili.com) - 视频网站

## 内部链接测试

以下链接应该不会触发安全提示（因为它们在白名单中）：

- [首页](/)
- [友链](/link)
- [归档](/archive)

## 功能说明

外链安全提示组件具有以下功能：

1. **自动拦截**：当用户点击外部链接时，自动触发安全提示弹窗
2. **白名单支持**：白名单内的域名不会触发安全提示
3. **安全警告**：显示明确的跳转警告信息
4. **用户选择**：提供"继续访问"和"取消"两个操作按钮
5. **新标签页打开**：点击"继续访问"后在新标签页打开目标链接

## 配置方式

在 `app/app.config.ts` 中可以配置白名单和开关：

```typescript
component: {
  externalLink: {
    enabled: true,
    whitelist: [
      'qxzhan.cn',
      'localhost',
      '127.0.0.1',
    ],
  },
}
```
