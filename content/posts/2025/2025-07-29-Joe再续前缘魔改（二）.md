## 这一期讲讲新增轻量朋友圈页面，需要提前部署[友链朋友圈](https://github.com/willow-god/Friend-Circle-Lite/)项目。

{callout color="#e64747"}
具体部署教程详见[Friend-Circle-Lite:轻量友链朋友圈](https://blog.liushen.fun/posts/4dc716ec/)
{/callout}

## 部署完成后，请在博客后台>管理>独立页面
::pic
---
src: https://qxzhan.cn/usr/uploads/2025/07/4201488640.png
caption: mdo185i4.png
---
::
#### 代码如下
```html
!!!
<div id="friend-circle-lite-root"></div>
<script>
    if (typeof UserConfig === 'undefined') {
        var UserConfig = {
            // 填写你的fc Lite地址
            private_api_url: 'https://example.com',
            // 点击加载更多时，一次最多加载几篇文章，默认20
            page_turning_number: 20,
            // 头像加载失败时，默认头像地址
            error_img: 'https://i.p-i.vip/30/20240815-66bced9226a36.webp',
        }
    }
</script>
<link rel="stylesheet" href="https://jsd.xnet.ren/gh/willow-god/Friend-Circle-Lite/main/fclite.min.css">
<script src="https://jsd.xnet.ren/gh/willow-god/Friend-Circle-Lite/main/fclite.min.js"></script>
!!!
```
